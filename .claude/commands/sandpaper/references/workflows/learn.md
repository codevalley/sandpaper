# Learn workflow

Append one callout described by the remaining user arguments to `brain/learnings.html`.

Use `<aside class="entry entry--learning callout gotcha|verdict" data-kind="learning"
data-status="active" data-date data-area data-tags id="l-…" data-cid="l-…">`. Include a bold
one-line claim, a sentence explaining why, and a backlink to where it occurred. Use `gotcha` for
something that broke and `verdict` for a settled judgment. Then execute the canonical `log` workflow.
