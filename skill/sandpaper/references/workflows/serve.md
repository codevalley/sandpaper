# Serve workflow

Run `node bin/cli.js brain/`, or `node bin/cli.js <doc.html>`, in the background. It serves at
`http://127.0.0.1:4848` with the on-page toolbar: **Sand** for a scoped AI edit, **✎ Hands** for a
direct edit without AI, and **❯ Sling** for a terminal-ready instruction. Tell the owner the URL and
that the file on disk remains the single source of truth.
