What is Sandpaper?

Sandpaper turns the rich HTML documents Claude Code makes for you into a living surface you refine by talking to the page — instead of reading the doc here and re-describing changes over there in a terminal, you change it in place, and it stays the canonical spec your build is based on.

The problem: the medium for thinking-with-Claude got richer (markdown → HTML with diagrams, tables, mockups), but the loop didn't — you still steer it from another window.
The bet: read the rich document and change it in place, with Claude rewriting the file as the single source of truth and showing live status.
